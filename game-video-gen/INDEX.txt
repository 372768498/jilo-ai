❓ 遇到问题?

   错误: Cannot find module
   → 运行: npm install

   错误: Unauthorized
   → API Key 已配置,检查网络

   错误: ECONNREFUSED
   → 检查网络连接

   错误: Generation failed
   → 访问 https://status.fal.ai/

   详细故障排查 → 查看 local_test_guide.md
