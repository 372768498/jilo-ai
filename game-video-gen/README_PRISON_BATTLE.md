## ❌ 常见问题

### Q1: 生成的视频不符合预期？

**A**: 这是AI生成，可能需要多次尝试：
```powershell
# 重新运行生成不同的视频
node test_prison_battle.js
```

每次生成都会有不同的结果。

### Q2: 想要不同的场景？

**A**: 修改 `test_prison_battle.js` 中的 prompt：

```javascript
prompt: `你的自定义场景描述（英文）`,
```

### Q3: 想要更长的视频？

**A**: SORA2 目前最长支持10秒。如需更长视频：
1. 生成多个10秒片段
2. 后期剪辑拼接

### Q4: 视频效果不够真实？

**A**: AI生成的视频是概念性的，建议：
1. 作为背景素材
2. 叠加真实游戏画面
3. 或作为创意参考
