## 事件经过：五角大楼合同争议

2026年2月27日，OpenAI宣布与美国国防部（五角大楼）建立合作伙伴关系，向其提供AI工具。此举立即引发用户、员工及公民社会组织的强烈批评，核心担忧是：这些工具可能被用于国内大规模监控和自主武器系统。

事件的爆发时机恰好助推了舆论浪潮：就在此前，五角大楼刚刚放弃了与Anthropic的AI合同——原因是Anthropic拒绝接受类似条款，理由是此类应用与民主价值观不相容。

早在2026年2月初，一场名为"QuitGPT"的草根运动便已兴起，号召用户取消订阅、卸载ChatGPT，以抗议其在安全和伦理方面的立场。五角大楼合同的曝光进一步推动了这场运动。

**直接影响：**
- ChatGPT 日卸载量暴增 295%（相较正常每日约 9% 的卸载率）
- ChatGPT 下载量：周六下降 13%，周日继续下降 5%
- Claude 下载量激增，登顶美国 App Store 榜首
- 近 900 名 OpenAI 及 Google 员工联署公开信，反对将 AI 用于国内大规模监控或缺乏人工监督的自主致命武器

2026年3月3日，OpenAI CEO 山姆·奥特曼承认此次合同"投机取巧且仓促"，"公关形象不佳"。OpenAI随即修订合同，明确禁止以下行为：
- 有意对美国公民和国民实施国内监控
- 未经单独合同修订，NSA使用OpenAI系统
- 利用AI模型指挥自主武器系统

## 1. Claude Pro — 写作与推理最佳选择

**Claude Pro 是 ChatGPT 用户迁移的首选目的地，2026年2月下旬登顶美国 App Store 下载榜。**

Claude Opus 4.6 在专业写作领域表现卓越——法律分析、高管沟通、长篇研究综述均有突出表现。人工评估者一致认为 Claude 在细腻、专家级别的文字表达上更胜一筹。测试版的100万 token 上下文窗口是 ChatGPT 40万 token 的2.5倍，可在单次对话中处理整本书、完整代码库或综合性法律合同。

在多学科推理方面，Claude 同样领先：MMLU 得分91%、ARC-AGI-2 得分68.8%，在《人类最后考试》（涵盖科学、法律、经济、人文）中成绩顶尖。在真实工程任务（SWE-Bench Verified）上，Claude 与 ChatGPT 并驾齐驱，双方均约为80%。

Anthropic 的伦理立场是用户迁移的关键因素。Anthropic 拒绝了 OpenAI 接受的五角大楼合同，理由是与民主价值观不相容。近900名 OpenAI 及 Google 员工联署公开信，支持这一立场。

**价格方案：** 免费（Sonnet 4.5，每5小时10~25条消息）→ Pro $20/月 → Max $100/月 → Max Pro $200/月

**最适合：** 专业写作者、法律/金融分析师、处理大型文档的研究人员，以及注重AI伦理发展的用户。

## 2. Gemini Advanced — 多模态任务最佳选择

**Gemini Advanced 是需要多模态 AI 并深度融入 Google 生态的用户的最佳 ChatGPT 替代品。**

Gemini 3.1 Pro（Gemini Advanced 的旗舰模型）支持在单次对话中处理文字、图像、视频和音频。Deep Search（深度搜索）可跨多来源综合研究，提供全面的信息整合。YouTube集成支持视频内容分析和摘要。Google Workspace 集成则打通了 Gmail、文档、表格和云端硬盘。

Gemini Advanced 附带2TB Google One 存储空间，对于已在 Google 生态内的用户而言，综合价值远超单独的 AI 订阅。在获得授权后，AI 可访问并分析您的邮件、文档和文件。

Gemini 3.1 Flash-Lite（2026年3月3日发布）是一款针对速度和成本优化的轻量模型，直接对标 GPT-4o-mini，可在免费版中用于基础任务。

**价格方案：** 免费（Gemini 3.1 Flash-Lite）→ Advanced $20/月（含2TB存储）

**最适合：** Google Workspace 用户、需要多模态分析的研究人员，以及希望将 AI 与云存储合二为一的用户。

## 3. Perplexity Pro — 研究调研最佳选择

**Perplexity Pro 是注重实时信息获取和带引用来源研究的用户的最佳 ChatGPT 替代品。**

Perplexity 的核心差异化优势是将实时网络搜索集成进每一个回答。与 ChatGPT 的知识截止日期不同，Perplexity 会搜索网络获取最新信息，并提供附带来源链接的引用式回答。这使其在研究调研、事实核查和追踪突发新闻或最新进展方面具有显著优势。

Perplexity Pro 提供无限量 Pro Search（深度研究模式）、图像生成和 API 访问权限。Pro Search 功能可对数十个来源进行多步推理，为复杂问题综合出全面答案。

Perplexity 不使用用户数据进行训练，为注重隐私的用户提供了 ChatGPT 的替代选择。该公司目前未宣布任何军事或国防合同。

**价格方案：** 免费（有次数限制）→ Pro $20/月（无限量 Pro Search）

**最适合：** 研究人员、记者、学生，以及需要可核实来源的实时信息的用户。

## 4. Microsoft Copilot Pro — Office 集成最佳选择

**Microsoft Copilot Pro 是企业用户和 Microsoft 365 订阅者的最佳 ChatGPT 替代品。**

Copilot Pro 直接集成到 Microsoft Word、Excel、PowerPoint、Outlook 和 Teams 中。它使用 GPT-5.2（与 ChatGPT 同款模型系列），但深度融合了微软生态。用户无需离开 Microsoft 应用，即可生成文档、分析电子表格、创建演示文稿和起草邮件。

Copilot Pro 在高峰时段提供 GPT-5.2 优先访问权、更快的响应速度，以及每天100次 Designer（AI 图像生成）提速额度。对于企业用户，适用于 Microsoft 365 的 Copilot（每用户每月$30）还提供商业数据保护和管理员控制功能。

微软虽有国防合同，但由于消费者产品与国防产品之间界限更为清晰，目前未面临与 OpenAI 同等程度的用户反弹。

**价格方案：** 免费（功能受限）→ Pro $20/月 → Microsoft 365 Copilot $30/用户/月

**最适合：** Microsoft 365 用户、企业团队，以及希望将 AI 集成进生产力工具的用户。

## 为什么用户正在离开 ChatGPT

295% 的卸载量激增反映了三个汇聚因素：

**1. 对军事用途的伦理担忧**
用户反对 OpenAI 与五角大楼的合作，担心 AI 被用于国内监控和自主武器。QuitGPT 运动将这一问题定义为一种选择：支持负责任 AI 发展（Anthropic 的立场），还是为潜在滥用推波助澜（OpenAI 最初签约的立场）。

**2. 竞争性替代品的成熟**
2024年，ChatGPT 还有明显的质量领先优势。到2026年，Claude、Gemini 和 Perplexity 在特定领域已提供与其相当甚至更优秀的能力。用户切换时不再面临显著的质量损失。

**3. 信任与透明度的缺失**
山姆·奥特曼承认合同"投机取巧且仓促"，损害了用户信任。合同的快速修订显示 OpenAI 最初将合同利益置于用户关切之上，之后在压力下才被迫改变。
